# 🌳 Forest Loss & Carbon Emissions Analysis (India)

## 📌 Project Overview
This project analyzes **forest loss and carbon emissions in India (2001–2023)** using data visualization techniques. It focuses on identifying trends, regional patterns, and the environmental impact of deforestation through an interactive dashboard built in Power BI.

The goal is to transform raw environmental data into meaningful insights that support **data-driven decision-making**.

---

## 🎯 Objectives
- Analyze forest loss trends over time  
- Study carbon emissions due to deforestation  
- Compare state-wise variations across India  
- Identify relationship between forest loss and emissions  
- Present insights using an interactive dashboard  

---

## 📊 Dataset
- **Source:** Global Forest Watch  
- **Data Includes:**
  - Forest Loss (hectares)
  - Carbon Emissions (metric tons)
  - Net Carbon Flux  
- **Time Period:** 2001 – 2023  

---

## 🛠️ Tools & Technologies
- **Microsoft Excel** – Data cleaning & preprocessing  
- **Power BI** – Data modeling & dashboard visualization  

---

## 🔄 Workflow
1. Data Collection from Global Forest Watch  
2. Data Cleaning (removing duplicates, handling missing values)  
3. Data Transformation (year-wise & state-wise aggregation)  
4. Dashboard Creation in Power BI  
5. Analysis & Interpretation of trends  

---

## 📈 Key Insights
- 📌 Forest loss in India shows an **increasing trend** over time  
- 📌 Carbon emissions have **risen alongside deforestation**  
- 📌 Northeastern states (Mizoram, Assam, Arunachal Pradesh) show **higher forest loss**  
- 📌 Strong **positive relationship** between forest loss and carbon emissions  
- 📌 Several regions are shifting from **carbon sinks to carbon sources**  

---

## 🧪 Hypothesis Testing
- **H₀:** No relationship between forest loss and carbon emissions  
- **H₁:** Significant relationship exists  

✅ Result: **H₀ rejected, H₁ accepted**  
→ Forest loss and carbon emissions are positively correlated  

---

## 📊 Dashboard Features
- Interactive filters (Year, State)  
- Line charts for trend analysis  
- Bar charts for state comparison  
- KPI cards for key metrics  
- Scatter plots for correlation analysis  

---

## 📌 Conclusion
The study highlights a **continuous rise in forest loss and carbon emissions in India**, indicating growing environmental concerns. It emphasizes the need for:
- Strong forest conservation policies  
- Sustainable land-use planning  
- Climate-focused decision-making  

The dashboard provides a **user-friendly analytical tool** for researchers and policymakers.
